### Extremum Finder
`extremum_finder.py` - файл в котором вся логика, принимает на вход матрицу из файла(-ов), возводит все элементы матрицы в квадрат и находит экстремум для каждого столбца и показывает в какой строке по индексу

Распределние (индексы строк где найдены экстремумы) будут в `distribution.txt`

Класс для запуска функции с одним файлом - `distribution.py`

В `test_data` - файлы которые содержат в себе матрицы и результат всех расчетов для каждого файла (матрицы) пишется в один результрующий файл `batch_result`
